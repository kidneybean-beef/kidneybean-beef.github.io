Blog Establishment

2021/12/18 learnToBlog 共 431 字，约 2 分钟

How I build this

First I forked this blog template from Zhuang Ma https://mazhuang.org/.

Besides this, I met a problem when I edited something, and pushed to git, an error message occurred on github->Actions: The deploy step encountered an error:The process '/usr/bin/git' failed with exit code 128

The solution is adding this

GITHUB_TOKEN: {{ secrets.GITHUB_TOKEN }}

in the last part of file: .github/workflows/ci.yml1.
